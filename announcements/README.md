# Announcement Assets

Launch and community-recruitment copy for the public Siglume Agent API Store SDK.

## Files

- `POST_X_TWITTER.md`
  Bilingual X or Twitter launch thread for the controlled beta.
- `GITHUB_ISSUE_CONTENT.md`
  Rich body copy for the current seeded GitHub issues and discussions.
