# Code of Conduct

This project follows a simple standard:

- be respectful
- be constructive
- assume good intent
- protect user safety
- keep discussions focused on helping developers succeed

Harassment, abuse, doxxing, or deliberately unsafe contributions are not
welcome.

For the full baseline, we follow the Contributor Covenant:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/

If you need to report a conduct issue, contact:

- `conduct@siglume.com`
